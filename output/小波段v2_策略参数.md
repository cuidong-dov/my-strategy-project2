## 一、策略定位

**恐慌低吸型小波段策略**。核心思想：在板块恐慌时买入、狂热时卖出，不做趋势追涨。

三种买入信号：
- **买入A（恐慌企稳）**：局部恐慌+缩量企稳，最核心信号
- **买入B（超卖缩量）**：超卖区域缩量，辅助信号
- **买入C（底背离）**：MACD/量价底背离+缩量，精准抄底

## 四、信号规则详解

### 4.1 买入A：恐慌企稳（主力信号）

**触发条件**（全部AND）：

```
1. 混合恐慌值 ≥ 55  （panic_mix = 局部恐慌×0.5 + 60日恐慌×0.5）
2. 局部板块风险 < 67
3. 大盘风险 < 74
4. 预估成功率 ≥ 40%
5. 缩量企稳（settle=1），或深度回调时放宽为"非放量下跌"
6. 量价质量 > -0.5
7. 从高点回调 ≥ 2%（除非已深度回调>15%）
8. 近5日放量下跌占比 < 0.5
```

**设计要点**：
- 恐慌值改用混合（局部+60日各50%），避免单日波动误触发
- 深度回调（>15%）时 settle 条件放宽：不要求缩量涨，只要求不是放量暴跌
- pullback_max 参数为0但代码强制要求≥2%回调，避免横盘买入

### 4.2 买入B：超卖缩量（辅助信号）

**触发条件**（全部AND）：

```
1. 超卖 ≥ 0.45
2. 缩量（vol_ratio < 1.0）
3. 全局板块风险 < 48
4. 大盘风险 < 74
5. 预估成功率 ≥ 40%
6. 量价质量 > -0.5
7. 从高点回调 ≥ 2%
```

### 4.3 买入C：底背离买入（精准信号）

**触发条件**（全部AND）：

```
1. MACD底背离 或 量价底背离
2. 全局板块风险 < 58
3. 大盘风险 < 74
4. 预估成功率 ≥ 35%（比A/B低5%）
5. 连续缩量 ≥ 2天
```

### 4.4 长线买入

**触发条件**：

```
1. 价格低于年线20%（disp_ma250 ≤ -0.20）
2. 板块风险 < 50
3. 大盘风险 < 80
4. 预估成功率 ≥ 43%
```

### 4.5 卖出信号

**触发条件**（任一OR）：

```
1. 追涨狂热 ≥ 76 AND 板块风险 ≥ 76
2. （MACD顶背离 OR 量价顶背离）AND 板块风险 ≥ 71
```

## 五、买点评分系统（1-10分）

每个买入信号生成后，自动进行7维度评分。各维度原始分1-10，加权求和后按板块内 min-max 归一化到1-10。

### 5.1 七维度详解

| 维度 | 权重 | 评分逻辑 |
|------|:---:|------|
| **回调深度** | 30% | 跌40%+=10分，跌30%+=9分，跌25%+=8分，跌20%+=7分，跌15%+=5.5分，跌12%+=4.5分，跌8%+=3分，跌5%+=2分，跌3%+=1.5分，几乎没跌=1分 |
| **背离确认** | 20% | MACD+量价双底背离=10分，MACD底背离=8分，量价底背离=6.5分，MACD金叉=4.5分，DIF转正=3分，无背离=1.5分 |
| **量价健康度** | 15% | 基准5分，量价质量>0.8加3分，settle加2分，缩量加0.5-1.5分，放量暴跌扣1.5-3分 |
| **趋势健康度** | 15% | 基准5分，多头排列加2分，空头排列扣2分，MA5斜率影响±1.5分，连跌≥5天扣2分 |
| **支撑位强度** | 10% | 双支撑=10分，强支撑=7.5分，均线附近=3.5-5分，无支撑=2分，跌破所有均线=1分 |
| **恐慌极端度** | 5% | 极度恐慌(>85)=10分，高度恐慌(>70)=8分，中度恐慌(>55)=6分，轻度恐慌(>40)=4分，低恐慌(>25)=2.5分，无恐慌=1.5分 |
| **企稳确认度** | 5% | 缩量4天+企稳=10分，缩量3天=8分，缩量2天=6分，企稳=4分，未企稳=2分 |

### 5.2 评分归一化

原始加权分（各维度×权重之和）在同一板块的所有买入信号间做 min-max 归一化：

```
归一化评分 = (原始分 - 板块最低分) / (板块最高分 - 板块最低分) × 9 + 1
```

**含义**：评分是板块内相对排名，10分=该板块历史上最好的买点，1分=最差的买点。

### 5.3 评分示例

```
2026-07-15 买入 [综合9.6分]
  回调深度：5.5分（跌15%+） → 背离确认：6.5分（量价底背离） → 
  量价健康度：5分（正常） → 趋势健康度：2.5分（空头排列） → 
  支撑位强度：7.5分（强支撑） → 恐慌极端度：1.5分（无恐慌） → 
  企稳确认度：2分（未企稳）
```

### 6.2 退出优先级

```
1. 卖出信号（最高优先级，与上一卖出信号间隔≥3天）
2. 硬止损 -4%
3. 移动止盈（peak回撤2.5%，需先浮盈1%）
4. 时间止损 22天
5. 期末平仓
```

### 6.3 优化评分函数

```
评分 = 7日准确率 × 0.35    ← 核心：买入信号短期准确度
     + 总收益归一化 × 0.20
     + 最大回撤归一化 × 0.15
     + 超额收益归一化 × 0.10
     + 胜率归一化 × 0.10
     + 交易次数归一化 × 0.10
```

## 九、v2.1 关键设计决策

| 决策 | 说明 |
|------|------|
| **无硬过滤** | 删除了"连续放量暴跌/MA空头加速/单日暴跌7%"三个硬过滤，风险通过评分表达 |
| **混合恐慌** | 恐慌值=局部×0.5+60日×0.5，避免单日波动误触发 |
| **深度回调放宽** | 回调>15%时settle只要求非放量下跌，不要求缩量涨 |
| **强制回调≥2%** | pullback_max=0但代码强制回调≥2%，避免横盘买入 |
| **评分归一化** | 板块内min-max归一化到1-10，表达相对排名 |
| **无趋势买点** | 纯低吸策略，不做趋势追涨/突破买入 |
