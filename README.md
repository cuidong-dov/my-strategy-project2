# A股板块评估系统 · 小波段策略

基于 **局部恐慌 + 缩量企稳 + 5日方向标定** 的 A 股板块小波段择时系统。

## 快速开始

```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 拉取最新数据（首次运行，约1分钟）
python scripts/01_fetch_data.py

# 3. 运行策略引擎（指标计算 + 两轮网格迭代，约40分钟）
python scripts/02_swing_engine.py

# 4. 生成 Excel 和 MD 报告
python scripts/03_build_output.py
```

## 输出文件

| 文件 | 说明 |
|------|------|
| `output/小波段策略_板块评估.xlsx` | 8个工作表：6板块六大字段 + 买卖点明细 + 策略参数 |
| `output/小波段_策略参数.md` | 策略参数与回测报告 |

## 评估板块

半导体 / 消费电子 / 通信 / 电池 / 电力 / 煤炭（同花顺行业指数，2021年至今）

## 六大字段

| 字段 | 含义 |
|------|------|
| 板块风险 | 板块过热/高波动程度（0-100） |
| 大盘风险 | 沪深300系统性风险（0-100） |
| 抄底狂热 | 放量逢跌抄底活跃度（0-100） |
| 追涨狂热 | 放量追高亢奋度（0-100） |
| 恐慌值 | 市场恐惧程度（0-100） |
| 买入预估成功率_5d | 历史同条件下5日后上涨概率（%） |

## 策略逻辑

**买入A（恐慌企稳）**：局部恐慌高位 + 缩量(量比<1) + 当日收涨 → 恐慌抛售后的惜售反弹

**买入B（超卖缩量）**：5日深度回撤 + 缩量 + 全局风险可控 → 跌不动了的反弹

**卖出**：追涨狂热 + 板块过热 → 亢奋止盈

**风控**：硬止损 4% | 移动止盈回撤 2.5%（盈利1%启用）| 时间止损 22日

## 定时运行

已配置 GitHub Actions，每个交易日北京时间 17:30 自动执行，输出文件自动提交回仓库。

也可以手动触发：GitHub → Actions → 每日小波段策略 → Run workflow。

## 迭代优化

策略参数通过两轮网格搜索优化：
- 迭代1：全局粗筛（~28,000 组合）
- 迭代2：最优邻域微调

评分权重：7日准确率 35% + 收益 20% + 回撤控制 15% + 超额 10% + 胜率 10% + 频率 10%

## 注意事项

- 数据源：同花顺行业指数（akshare）
- 回测区间：2021-01-04 至今（in-sample），部署前建议 walk-forward 验证
- 指数与可交易 ETF 存在跟踪误差，实盘需映射到具体标的
